Downloading a static webpage with python

  1. 05 août 2018
  2. Python
  3. Web Scraping

If you are using python legacy (aka python 2) first of all, stop ! Furthermore, this method won't work in python legacy

# Import modules
from urllib.request import urlopen

The webpage source code can be downloaded with the command urlopen

url = "http://example.com/"
#create a HTTP request in order to read the page
page = urlopen(url).read()

The source code will be stored in the variable page as a string 

print(page)
